Política de Privacidade

Última atualização: junho de 2026

1. Introdução

O InvestMap é um aplicativo de gestão de carteira de investimentos desenvolvido por Nespoli Digital. Esta Política de Privacidade descreve como tratamos suas informações ao usar nosso aplicativo.

2. Dados que Coletamos

O InvestMap armazena seus dados de investimentos localmente no seu dispositivo. As operações registradas, posições da carteira, dividendos e métricas financeiras ficam apenas no seu aparelho.

Dados transmitidos externamente:

  • Análises de IA: Quando você usa o assistente de análise, um resumo da sua carteira (sem dados pessoais identificáveis) é enviado à API do Google Gemini.
  • Cotações de ativos: O aplicativo consulta APIs públicas de cotações para exibir preços atualizados. Nenhum dado seu é enviado nessas consultas.
  • Google AdMob: O AdMob pode coletar dados de uso e identificadores de dispositivo para personalização de anúncios (usuários gratuitos).
  • Google Play Billing: Dados de compra processados pelo Google para assinaturas premium.

3. Como Usamos os Dados

  • Para registrar e exibir suas operações e posições de carteira
  • Para calcular métricas de rentabilidade e dividendos
  • Para fornecer análises de IA sobre sua carteira
  • Para exibir cotações atualizadas dos seus ativos
  • Para exibir anúncios (usuários gratuitos)

4. Compartilhamento de Dados

Não vendemos ou compartilhamos seus dados pessoais. Integrações com terceiros:

  • Google Gemini API: Análise de carteira anonimizada
  • Google AdMob: Exibição de anúncios
  • Google Play: Processamento de assinaturas
  • APIs de cotações: Consultas de preços de ativos (sem dados do usuário)

5. Segurança

Seus dados de investimentos ficam protegidos pelo sistema de armazenamento seguro do Android. O aplicativo não transmite sua carteira completa para nenhum servidor externo.

6. Seus Direitos

  • Acesso e visualização: Todos os dados disponíveis diretamente no app
  • Exportação: Exporte sua carteira em CSV nas Configurações
  • Exclusão: Delete todos os dados nas Configurações do app

7. Aviso de Investimentos

O InvestMap é uma ferramenta de organização e não constitui consultoria financeira ou recomendação de investimentos. As análises de IA são informativas e não devem ser usadas como única base para decisões financeiras.

8. Crianças

O InvestMap não é destinado a menores de 18 anos e não coletamos dados de crianças.

9. Alterações nesta Política

Esta política pode ser atualizada. Mudanças significativas serão comunicadas via atualização do aplicativo.
