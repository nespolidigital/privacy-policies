Política de Privacidade

Última atualização: junho de 2026

1. Introdução

O FitMap é um aplicativo de treino desenvolvido por Nespoli Digital. Esta política descreve como tratamos suas informações de saúde e atividade física ao usar o aplicativo.

2. Dados que Coletamos

O FitMap armazena seus dados de treino localmente no seu dispositivo. Fichas de treino, sessões, biometria e recordes pessoais ficam apenas no seu aparelho.

Dados transmitidos externamente:

  • Personal Trainer IA: Quando você usa o assistente de treino, informações básicas do seu perfil (objetivo, nível de experiência) e histórico resumido são enviados à API do Google Gemini. Dados biométricos sensíveis não são incluídos.
  • Biblioteca de exercícios: O aplicativo consulta uma API pública de exercícios para exibir novos movimentos. Nenhum dado seu é enviado nessas consultas.
  • Google AdMob: O AdMob pode coletar dados de dispositivo para personalização de anúncios (usuários gratuitos).
  • Google Play Billing: Dados de compra processados pelo Google para assinaturas premium.

3. Dados de Saúde

O FitMap pode coletar dados relacionados à saúde como peso, altura, medidas corporais e histórico de treinos. Esses dados são considerados sensíveis e são armazenados exclusivamente no seu dispositivo, nunca sendo compartilhados com terceiros sem seu consentimento.

4. Como Usamos os Dados

  • Para registrar e organizar seus treinos e fichas
  • Para calcular métricas de progresso e recordes pessoais
  • Para fornecer sugestões personalizadas via Personal Trainer IA
  • Para exibir exercícios da biblioteca
  • Para exibir anúncios (usuários gratuitos)

5. Compartilhamento de Dados

Não vendemos seus dados de saúde ou treino. Integrações com terceiros:

  • Google Gemini API: Perfil básico anonimizado para o Personal Trainer IA
  • ExerciseDB API: Consulta de exercícios (sem dados do usuário)
  • Google AdMob: Exibição de anúncios
  • Google Play: Processamento de assinaturas

6. Segurança

Seus dados de saúde e treino são armazenados de forma segura no seu dispositivo. O aplicativo roda integralmente em modo offline para as funcionalidades principais.

7. Seus Direitos

  • Acesso: Todos os dados visíveis no app
  • Exportação: Exporte seu histórico de treinos nas Configurações
  • Exclusão: Delete todos os dados nas Configurações do app

8. Aviso de Saúde

O FitMap é uma ferramenta de organização de treinos e não substitui orientação médica ou de educador físico profissional. Consulte um profissional de saúde antes de iniciar qualquer programa de exercícios.

9. Crianças

O FitMap não é destinado a menores de 13 anos. Não coletamos intencionalmente dados de crianças.

10. Alterações nesta Política

Esta política pode ser atualizada. Mudanças serão comunicadas via atualização do aplicativo.
