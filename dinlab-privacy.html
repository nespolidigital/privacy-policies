Política de Privacidade

Última atualização: junho de 2026

1. Introdução

O DinLab é um aplicativo de finanças pessoais desenvolvido por Nespoli Digital. Esta Política de Privacidade descreve como coletamos, usamos e protegemos suas informações ao usar nosso aplicativo.

2. Dados que Coletamos

O DinLab opera predominantemente de forma offline. Os dados financeiros inseridos no aplicativo (transações, categorias, metas, orçamentos) são armazenados localmente no seu dispositivo e não são enviados para nossos servidores.

Os únicos dados transmitidos externamente são:

  • Análises de IA: Quando você utiliza o recurso de análise financeira por inteligência artificial, um resumo anonimizado dos seus dados financeiros é enviado para a API do Google Gemini para processamento. Nenhum dado pessoal identificável é incluído.
  • Dados de anúncios: O aplicativo utiliza o Google AdMob para exibição de anúncios na versão gratuita. O AdMob pode coletar dados de uso e identificadores de dispositivo conforme sua própria política de privacidade.
  • Dados de assinatura: Informações de compra são processadas pelo Google Play Billing. Não armazenamos dados de pagamento.

3. Como Usamos os Dados

  • Para fornecer funcionalidades do aplicativo (registro de transações, relatórios, metas)
  • Para processar análises financeiras via inteligência artificial
  • Para exibir anúncios relevantes (usuários gratuitos)
  • Para processar assinaturas premium

4. Compartilhamento de Dados

Não vendemos, alugamos ou compartilhamos seus dados pessoais com terceiros, exceto:

  • Google Gemini API: Para processamento de análises de IA (dados anonimizados)
  • Google AdMob: Para exibição de anúncios (conforme política do Google)
  • Google Play: Para processamento de assinaturas

5. Segurança dos Dados

Seus dados financeiros são armazenados localmente no seu dispositivo com proteção do sistema operacional Android. Recomendamos manter seu dispositivo protegido com senha ou biometria.

6. Seus Direitos

  • Acesso: Todos os seus dados estão acessíveis diretamente no aplicativo
  • Exclusão: Você pode excluir todos os dados nas Configurações do app
  • Exportação: Você pode exportar seus dados em formato JSON ou CSV nas Configurações

7. Retenção de Dados

Os dados são armazenados no seu dispositivo até que você os exclua manualmente ou desinstale o aplicativo. Não mantemos cópias dos seus dados em nossos servidores.

8. Crianças

O DinLab não é direcionado a crianças menores de 13 anos e não coletamos intencionalmente dados de crianças.

9. Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Notificaremos sobre mudanças significativas por meio de uma atualização no aplicativo.
