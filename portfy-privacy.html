Política de Privacidade

Última atualização: junho de 2026

1. Introdução

O Portfy é um aplicativo de portfólio profissional desenvolvido por Nespoli Digital. Esta política descreve como tratamos suas informações pessoais e profissionais ao usar o aplicativo.

2. Dados que Coletamos

O Portfy armazena seus dados localmente no seu dispositivo. Projetos, clientes, serviços e configurações do portfólio ficam apenas no seu aparelho.

Dados transmitidos externamente:

  • Assistente de IA: Quando você usa o assistente para sugestões de portfólio ou análise de projetos, um contexto resumido é enviado à API do Google Gemini. Dados de clientes não são incluídos.
  • Google AdMob: O AdMob pode coletar dados de dispositivo para personalização de anúncios (usuários gratuitos).
  • Google Play Billing: Dados de compra processados pelo Google para assinaturas premium.

3. Dados Profissionais

Informações sobre seus projetos, clientes e trabalhos são consideradas dados profissionais sensíveis. O Portfy não acessa, transmite ou compartilha essas informações com terceiros. Todo o conteúdo do seu portfólio fica exclusivamente no seu dispositivo.

4. Como Usamos os Dados

  • Para organizar e exibir seus projetos e portfólio
  • Para gerar análises e sugestões via inteligência artificial
  • Para exibir anúncios (usuários gratuitos)
  • Para processar assinaturas premium

5. Compartilhamento de Dados

Não vendemos seus dados. Integrações com terceiros:

  • Google Gemini API: Contexto anonimizado para o assistente de IA
  • Google AdMob: Exibição de anúncios
  • Google Play: Processamento de assinaturas

6. Segurança

Seus dados profissionais ficam protegidos pelo sistema de armazenamento seguro do Android. Recomendamos fazer backups regulares usando a função de exportação do aplicativo.

7. Seus Direitos

  • Acesso: Todos os dados visíveis e editáveis diretamente no app
  • Exportação: Exporte seu portfólio em JSON nas Configurações
  • Exclusão: Delete todos os dados nas Configurações do app
  • Portabilidade: Importe dados de backup em outros dispositivos

8. Crianças

O Portfy é destinado a profissionais e freelancers adultos. Não coletamos dados de menores de 13 anos.

9. Alterações nesta Política

Esta política pode ser atualizada. Mudanças serão comunicadas via atualização do aplicativo.
